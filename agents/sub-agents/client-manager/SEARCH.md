# Поиск заказов — Client Manager

## 🚀 Быстрый старт

### Запуск поиска заказов

```bash
npm run search
```

Скрипт найдёт тестовые заказы и выведет их на экран.

---

## 📋 Как это работает

### 1. Поиск заказов

Скрипт `search-demo.js` генерирует тестовые заказы с различных бирж:
- FL.ru
- Freelance.ru
- Kwork.ru
- Weblancer.net
- Habr Freelance
- Telegram-каналы

### 2. Вывод на экран

Заказы отображаются в удобном формате:
- Название проекта
- Бюджет
- Ссылка
- Краткое описание
- Тип проекта (wordpress, bitrix, landing, etc.)

### 3. Импорт в CRM

После поиска скрипт предложит импортировать заказы в CRM:
```
📥 Импортировать заказы в CRM? (y/n): y
```


## ⚠️ Важные замечания

### Яндекс-парсинг
Яндекс блокирует автоматический парсинг. В текущей версии используется демо-режим с тестовыми данными.

### Реальный парсинг
Для реального парсинга используйте:
1. **Telegram-каналы** — нужен API ключ (https://my.telegram.org/apps)
2. **Kwork** — нужна авторизация (email/password в .env)

---

## 🎯 Следующие шаги

### 1. Настройте Telegram API (опционально)

Получите ключи на https://my.telegram.org/apps и добавьте в `.env`:

```env
TELEGRAM_API_ID=12345678
TELEGRAM_API_HASH=abcdef1234567890
```

### 2. Запустите полный парсинг

```bash
npm run parse-leads
```

### 3. Используйте CRM

После импорта заказов используйте Client Manager для:
- Генерации коммерческих предложений
- Управления статусами клиентов
- Напоминаний о follow-up

## 🐛 Решение проблем

### Ошибка при импорте в CRM
Убедитесь, что зависимости установлены:
```bash
npm install
```

### Ничего не найдено
Используйте демо-режим:
```bash
npm run search
```
