# Client Manager — Поиск заказов

## 🚀 Быстрый старт

```bash
npm run search
```

---

## 📋 Возможности

### 1. Поиск заказов
- Демо-режим с тестовыми данными
- 8 заказов с различных бирж
- Группировка по источникам

### 2. Вывод на экран
- Красивое форматирование
- Бюджет, ссылка, описание
- Тип проекта (wordpress, bitrix, landing)

### 3. Импорт в CRM
- Автоматическое добавление лидов
- Присвоение статусов
- Сохранение истории

## 🎯 Использование

### Команды

```bash
# Поиск заказов (демо)
npm run search

# Тест CRM
npm run client-manager

# Полный парсинг (нужны API ключи)
npm run parse-leads
```

### Автоматический импорт

Для автоматического импорта без вопроса измените `search-demo.js`:

```javascript
// Вместо:
const answer = await askQuestion('📥 Импортировать заказы в CRM? (y/n): ');
if (answer.toLowerCase() === 'y') { ... }

// Используйте:
await importToCRM(orders); // Авто-импорт
```
