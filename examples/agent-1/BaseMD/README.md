# BaseMD - База знаний AI-агента

## Структура

```
BaseMD/
├── context/          # Контекстные файлы для понимания задач
│   └── .gitkeep
├── tasks/            # История выполненных задач и результаты
│   └── .gitkeep
└── memory/           # Долговременная память агента
    └── .gitkeep
```

## Описание категорий

### context/
Файлы контекста содержат информацию о текущих задачах, требованиях и условиях выполнения.

**Формат файла:**
```markdown
# Контекст: [Название]

**Теги:** [теги]
**Дата создания:** [ISO дата]

---

[Содержимое контекста]

---
*Сгенерировано автоматически BaseMD*
```

### tasks/
Файлы задач содержат историю выполненных задач с результатами.

**Формат файла:**
```markdown
# Задача: [Описание]

**ID задачи:** [task-id]
**Статус:** completed/failed
**Дата выполнения:** [ISO дата]

---

## Результат

[Результат выполнения]

---
*Сгенерировано автоматически BaseMD*
```

### memory/
Файлы памяти содержат долгосрочные знания и важную информацию.

**Формат файла:**
```markdown
# Память: [Название]

**Важность:** ★★★☆☆
**Дата записи:** [ISO дата]

---

[Содержимое памяти]

---
*Сгенерировано автоматически BaseMD*
```

## API использования

```javascript
import { BaseMDManager } from '../../shared/base-md.js';

const baseMD = new BaseMDManager('agent-id', './BaseMD');
await baseMD.initialize();

// Добавление контекста
await baseMD.addContext('Название', 'Содержимое', ['тег1', 'тег2']);

// Добавление задачи
await baseMD.addTask('task-1', 'Описание задачи', 'Результат', 'completed');

// Добавление памяти
await baseMD.addMemory('Важное знание', 'Содержимое', 4);

// Поиск
const results = await baseMD.search('ключевое слово');

// Чтение файла
const content = await baseMD.read('context', 'filename.md');
```

## Уровни важности памяти

- ★☆☆☆☆ - Низкая важность (временная информация)
- ★★☆☆☆ - Ниже средней (вспомогательные данные)
- ★★★☆☆ - Средняя важность (обычные знания)
- ★★★★☆ - Высокая важность (критичные знания)
- ★★★★★ - Критическая важность (основополагающие принципы)
